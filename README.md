# Dashboard de Análise de Dados - Imersão Alura


É um dashboard interativo desenvolvido com **Streamlit** para analisar a distribuição salarial de cargos na area de dados, em dolar.

## Tecnologias
* Python
* Streamlit
* Pandas 
* Ploty

## O Dataset
O arquivo `dados_imersao.csv` contém informações que tratamos durante os dias da imersão de dados **Alura**.

## Como rodar o projeto
1. Instale as dependências:
   No seu terminal com ->  pip install -r requirements.txt
